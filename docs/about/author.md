# 作者与项目

UIChat Mira 由 **Tomz Dang** 发起并持续构建。

- 当前产品版本：`0.7.1`

## 一个人也要建立秩序

Mira 的源码不是一份简单 demo。仓库同时维护桌面壳、renderer、本地后端、数据库、Agent runtime、Harness、知识系统和大量验证材料。

这也解释了项目为什么如此重视契约、任务卡、评审证据与文档真相：当系统复杂到无法靠一个人记住全部细节时，结构本身必须承担记忆。
