# UIChat Mira

> 让模型成为真正可工作的本地智能体。

UIChat Mira 是面向桌面与独立 Web 场景的多模型 AI 工作台。它把模型、知识、工具与权限放进一个可观察、可恢复、可演进的执行系统。

## 为什么做 Mira

Mira 的目标不是成为一个更花哨的聊天壳，而是成为一个有人使用、能够长期维护的产品。

| 原则 | 含义 |
| --- | --- |
| Multi-provider | 产品能力不绑定单一模型供应商 |
| Local-first | 桌面端是核心，Web 可独立运行 |
| Evidence-driven | 证据驱动执行，关键判断可复核 |
