# 一个人带着一群 Agent 工作，到底改变了什么

Agent 让独立开发第一次像拥有团队，但也让协调本身变成新的工作。

## 变化最明显的地方

- 我可以并行推进更多事
- 我更需要清楚地描述目标和边界
- 我必须花心力判断哪些工作该交给谁

## 仍然没有改变的

最后做判断的人，还是我。
