# Windows 文件链接为什么总在最不该坏的时候坏

文件链接看似小问题，坏一次就是用户信任断一次。

## 这类问题通常怎么出现

- 临时路径和工作区路径混用
- 空格、中文和短路径处理不一致
- 渲染层和执行层对路径语义理解不同

## 这次统一了什么

- 全部本地链接都输出绝对路径
- 显示文本和真实跳转目标分离
- 临时附件进入公开资源前先做落盘
