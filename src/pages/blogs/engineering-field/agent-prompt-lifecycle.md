# Agent 提示词的生命周期，应该和任务一起被管理

提示词不是开场白，它是任务执行过程的一部分。

## 老问题出在哪里

- 一个提示覆盖全部阶段，信息密度失衡
- 进入编码、验证、收尾时需要的约束完全不同
- 模型常常被迫在一大段静态规则里自己找重点

## 这次的方向

- 把提示拆成阶段片段
- 在任务切换时注入不同的约束
- 保留全局规则，但减少阶段噪音
