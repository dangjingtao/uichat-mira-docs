# v0.7：工作台记忆终于开始像记忆了

这次版本不是加一个按钮，而是重写我们对“记住用户正在做什么”的理解。

## 这次为什么要改

- 旧版本只能记住对话片段，记不住任务状态
- 用户回到工作台后，经常要重新解释背景
- 记忆如果不和任务进度绑定，就只是更长的聊天记录

## 我们改了什么

- 用任务槽位替代单纯消息缓存
- 把最近决策、待办和阻塞拆开存
- 回到会话时先还原任务状态，再还原上下文

## 还有哪些没做完

我们还没有把“长期记忆”和“当前任务记忆”彻底分层，这会是下一轮要继续拆的地方。
